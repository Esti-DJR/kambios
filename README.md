=== KAMBIOS ===

A very small, very basic, CLI + GUI (PyQt6) file renamer, done in Python.

This is just a basic file renamer that does three things:

1. Number files (for example: 1 - My Book.pdf, 2 - My book.pdf, 3 - My book.pdf)
2. Replace whole names (for example: My movie.mp4, My movie.srt)
3. Replace parts of names (for example: Changing "My Game (USA v2.0).zip" to "My Game.zip")

And that's pretty much it.
